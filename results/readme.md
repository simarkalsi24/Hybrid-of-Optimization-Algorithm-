# Results and Analysis

This folder contains all the outputs generated during the experiments, including evaluation metrics, visualizations, and logs. It provides a complete view of how the models performed under different hybrid optimization strategies.

---

## What this folder includes

### Figures
This section contains visual outputs from the training and evaluation process, such as:

- Accuracy and loss curves  
- Confusion matrices  
- Performance comparison plots  

These plots help in understanding how the model learns over time and how well it generalizes.

---

### CSV Files

This section includes structured data generated during experiments, such as:

- Training logs  
- Hyperparameter search results  
- Performance summaries  

These files can be used for further analysis or reproducibility.

---

### Classification Report and Summary

This contains the detailed evaluation metrics of the trained models, including:

- Precision  
- Recall  
- F1-score  
- Overall accuracy  

These metrics give a clearer picture of model performance beyond just accuracy.

---

### HybridResults.ipynb

This notebook is used for analyzing and visualizing the results.

It combines:
- Model outputs  
- Performance metrics  
- Graphs and comparisons  

This serves as a central place to interpret the outcomes of all hybrid approaches.

---

## Key observations

- Hybrid optimization methods generally perform better than single algorithms  
- Some combinations show faster convergence during training  
- Performance improvements are consistent across different evaluation metrics  
- The optimized models show strong generalization on test data  

---

## Notes

- All results are based on the same dataset and preprocessing pipeline  
- Minor variations may occur due to randomness in training and optimization  
- Figures and logs are directly generated from the training scripts
