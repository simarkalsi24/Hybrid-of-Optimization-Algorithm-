## Dataset

This project uses the Tomato Leaf Disease Dataset V2 available on Kaggle:

🔗 https://www.kaggle.com/datasets/zunorain/tomato-dataset-v2

Due to GitHub size limitations, the dataset is not included in this repository.

### Instructions:
1. Download the dataset from Kaggle
2. Extract it into the `data/` directory
3. Run the training script
